## App Frontend

1. Prerequisite

   - Install NodeJS 21
   - Install visual studio code
   - Install Docker

1. Build & Run Application

## Run folder appFE

```bash
   npm install pnpm -g
   pnpm install
   pnpm dev
```

## Run with docker

```bash
    docker build -t your_image_name:latest .
    docker run - your_image_name:latest
```
